# Ahmed Ashraf - Portfolio Website

A modern, responsive portfolio website featuring a purple theme with smooth scrolling animations.

## Features

- **Single Page Design** - Easy navigation with smooth scrolling
- **Responsive Layout** - Looks great on all devices
- **Animated Sections** - Engaging content reveal animations using AOS library
- **Project Filtering** - Filter projects by category
- **Contact Form** - Get in touch easily
- **Purple Theme** - Modern and visually appealing design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll) library
- Font Awesome Icons

## Setup Instructions

1. Clone this repository or download the files
2. Place your profile image named `bas.png` in the root directory
3. Open `index.html` in your web browser to view the site

## Customization

- **Colors**: Edit the CSS variables in `styles.css` to change the color scheme
- **Content**: Update your personal information, projects, and skills in `index.html`
- **Image**: Replace `bas.png` with your own profile image

## Structure

- `index.html` - Main HTML file
- `styles.css` - All styles and animations
- `script.js` - JavaScript functionality
- `bas.png` - Profile image

## Deployment

This site can be deployed on any web server or hosting service like GitHub Pages, Netlify, or Vercel.
